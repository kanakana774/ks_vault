### 単元 1. データベースの検索/登録/更新/削除/関数

**問題 1**
`artist` テーブルから、すべてのアーティストの `name` を取得してください。

**回答**

```sql
SELECT name FROM artist;
```

---

**問題 2**
`track` テーブルから、`unit_price` が `0.99` ドルであるすべてのトラックの `name` と `unit_price` を取得してください。

**回答**

```sql
SELECT name, unit_price FROM track WHERE unit_price = 0.99;
```

---

**問題 3**
`customer` テーブルから、`country` が 'USA' の顧客の `first_name`, `last_name`, `email` を取得してください。

**回答**

```sql
SELECT first_name, last_name, email FROM customer WHERE country = 'USA';
```

---

**問題 4**
`invoice` テーブルから、`total` が `10.00` より大きいすべての請求書の `invoice_id`, `customer_id`, `total` を取得してください。

**回答**

```sql
SELECT invoice_id, customer_id, total FROM invoice WHERE total > 10.00;
```

---

**問題 5**
`employee` テーブルに新しい従業員を登録してください。
`first_name` は 'John', `last_name` は 'Doe', `title` は 'IT Support' とします。他の情報は NULL で構いません。

**回答**

```sql
INSERT INTO employee (first_name, last_name, title)
VALUES ('John', 'Doe', 'IT Support');
```

---

**問題 6**
`album` テーブルで、`album_id` が `1` のアルバムの `title` を 'For Those About To Rock (Remastered)' に更新してください。

**回答**

```sql
UPDATE album
SET title = 'For Those About To Rock (Remastered)'
WHERE album_id = 1;
```

---

**問題 7**
`genre` テーブルから、`genre_id` が `25` のジャンルを削除してください。

**回答**

```sql
DELETE FROM genre
WHERE genre_id = 25;
```

---

**問題 8**
`track` テーブルから、`name` に 'Love' を含むすべてのトラックの `track_id`, `name`, `composer` を取得してください。大文字小文字は区別しないものとします。

**回答**

```sql
SELECT track_id, name, composer FROM track WHERE name ILIKE '%Love%';
```

**別パターン (大文字小文字を区別する場合)**

```sql
SELECT track_id, name, composer FROM track WHERE name LIKE '%Love%';
```

---

**問題 9**
`invoice` テーブルから、2021 年 1 月中に発行された請求書の `invoice_id`, `invoice_date`, `total` を取得してください。

**回答**

```sql
SELECT invoice_id, invoice_date, total
FROM invoice
WHERE invoice_date >= '2021-01-01' AND invoice_date < '2021-02-01';
```

**別パターン (BETWEEN を使用)**

```sql
SELECT invoice_id, invoice_date, total
FROM invoice
WHERE invoice_date BETWEEN '2021-01-01' AND '2021-01-31 23:59:59';
```

**別パターン (DATE_TRUNC 関数を使用)**

```sql
SELECT invoice_id, invoice_date, total
FROM invoice
WHERE DATE_TRUNC('month', invoice_date) = '2021-01-01';
```

---

**問題 10**
`customer` テーブルから、`country` が 'USA' または 'Canada' の顧客の `first_name`, `last_name`, `country` を取得してください。

**回答**

```sql
SELECT first_name, last_name, country
FROM customer
WHERE country = 'USA' OR country = 'Canada';
```

**別パターン (IN 演算子を使用)**

```sql
SELECT first_name, last_name, country
FROM customer
WHERE country IN ('USA', 'Canada');
```

---

**問題 11**
`track` テーブルから、`milliseconds` が `300000` (5 分) 以上のトラックの数をカウントしてください。

**回答**

```sql
SELECT COUNT(*)
FROM track
WHERE milliseconds >= 300000;
```

---

**問題 12**
`artist` テーブルから、アーティスト名をアルファベット順に並べ替えて取得してください。

**回答**

```sql
SELECT name FROM artist ORDER BY name ASC;
```

**別パターン (ASC を省略)**

```sql
SELECT name FROM artist ORDER BY name;
```

---

**問題 13**
`customer` テーブルから、`phone` が登録されていない（NULL である）顧客の `first_name`, `last_name`, `email` を取得してください。

**回答**

```sql
SELECT first_name, last_name, email
FROM customer
WHERE phone IS NULL;
```

---

**問題 14**
`track` テーブルから、`unit_price` が `0.99` でないトラックの `name` と `unit_price` を取得してください。

**回答**

```sql
SELECT name, unit_price
FROM track
WHERE unit_price != 0.99;
```

**別パターン (!=の代わりに<>を使用)**

```sql
SELECT name, unit_price
FROM track
WHERE unit_price <> 0.99;
```

---

**問題 15**
`employee` テーブルから、`birth_date` の年が 1970 年以前の従業員の `first_name`, `last_name`, `birth_date` を取得してください。

**回答**

```sql
SELECT first_name, last_name, birth_date
FROM employee
WHERE EXTRACT(YEAR FROM birth_date) <= 1970;
```

**別パターン (比較演算子を使用)**

```sql
SELECT first_name, last_name, birth_date
FROM employee
WHERE birth_date <= '1970-12-31';
```
