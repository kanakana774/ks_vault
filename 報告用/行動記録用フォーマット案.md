#### ② 月次サマリー報告書

デイリーログを基に、月に一度、上長や人事向けにサマリーを作成します。この報告書が、リスク提示と採用フィードバックの直接的な材料となります。

**件名：** 研修月次報告書（対象者：〇〇 〇〇）
**報告日：** 202X年XX月XX日
**報告者：** (あなたの氏名)

**1. 研修参加状況サマリー（定量的データ）**

| 項目 | 実績 | 備考 |
| :--- | :--- | :--- |
| **勤怠** | 遅刻：`X`回 / 欠席：`Y`回 | |
| **演習課題提出率** | `XX` % （提出 `X`回 / 全 `Y`回） | 期限内提出率：`XX` % |
| **演習課題平均完成度** | `XX` % | 講師によるレビュー結果の平均 |

**2. 研修中の行動特性（定性的サマリー）**

*   **指示理解力：** 講義で説明済みの内容に関する質問が頻発する傾向あり（月`X`回）。口頭での指示が一度で伝わらない場面が散見される。
*   **自走力/問題解決能力：** エラー発生時に自己解決を試みる姿勢が見られず、即座に他者（講師）に頼る傾向が強い。
*   **協調性/コミュニケーション：** グループワークにおいて、他者の意見を否定する発言が多く、チーム内の議論を停滞させる要因となることがあった（`X`月`Y`日のグループワーク等）。
*   **指導への反応：** フィードバックに対し、「でも」「だって」と反論することが多く、素直に受け入れる姿勢に課題が見られる。

**3. 予測されるリスク分析**

本人の行動特性に基づき、このまま研修を継続、あるいは現場へ配属された場合に想定されるリスクを具体的に記載します。

*   **【品質・生産性に関するリスク】**
    *   指示理解の低さから、要件の誤認や手戻りが多発し、プロジェクト全体の生産性を低下させる可能性がある。
    *   自走力の欠如により、OJT担当者の工数を過度に奪い、チーム全体のパフォーマンスが低下する恐れがある。
*   **【対人・組織に関するリスク】**
    *   協調性の低さや他責傾向が、チーム内の人間関係を悪化させ、他のメンバーのモチベーションを低下させる可能性がある。
    *   顧客との会議において、不適切な発言で信頼関係を損なうリスクがある。
*   **【本人に関するリスク】**
    *   現状のスキル・特性では現場での業務遂行が困難であり、本人が精神的に追い詰められ、早期離職やメンタル不調につながる可能性が高い。

**4. 人事採用業務へのフィードバック**

今回のケースを踏まえ、採用ミスマッチを防ぐために、採用選考で確認すべき観点を提言します。

*   **【確認すべきスキル・経験】**
    *   **エラー解決経験：** 「過去に解決した最も困難なエラーは何か、どうやって解決したか」という質問で、自走力や問題解決プロセスを確認する。
    *   **チーム開発経験：** チームでの役割や、意見が対立した際の対処法について具体的なエピソードを聞き、協調性を評価する。
*   **【見極めるべきパーソナリティ】**
    *   **素直さ/傾聴力：** 面接官からのフィードバックや、少し意地の悪い質問に対する反応を見て、素直に物事を受け入れられるかを評価する。
    *   **ストレス耐性：** 困難な状況やプレッシャーがかかった経験について質問し、他責にせず、状況を客観的に捉えて乗り越えようとする姿勢があるかを確認する。

### ★★★ 記録・報告する際の最重要注意点 ★★★

1.  **主観や感情を徹底的に排除する**
    *   **NG例：** 「やる気がなく、態度が悪い」
    *   **OK例：** 「演習時間中に5回あくびをし、スマートフォンを操作していたため注意した」

2.  **「事実」と「自分の解釈」を切り分ける**
    *   **事実：** グループワーク中、一度も発言しなかった。
    *   **解釈：** 彼は非協力的だ。
    *   報告書には「事実」のみを記載します。「解釈」は相手（上長や人事）が下すものです。

3.  **行動は「5W1H」で具体的に記述する**
    *   いつ（When）、どこで（Where）、誰が（Who）、何を（What）、なぜ（Why）、どのように（How）。これを意識すると、記録の客観性が格段に上がります。

4.  **改善指導の記録も必ずセットで残す**
    *   問題行動だけを記録すると、「講師は指導を尽くしたのか？」と指摘されかねません。「〇〇という問題行動に対し、△△と指導したが、改善が見られなかった」という記録を残すことで、あなたの責任問題になることを防ぎ、相手が改善困難な人材であることの証明になります。

5.  **一人で抱えず、定期的に共有する**
    *   問題が深刻化する前に、このフォーマットを使って上長や関係者にこまめに情報共有しましょう。あなた一人が「問題児」のレッテルを貼っているのではなく、組織としての客観的な評価であるという状況を作ることが重要です。
