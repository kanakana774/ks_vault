### ★フォーマットの特長と使い方★
- **1日1行で完結:** このファイルを開き、テーブルの最終行にその日の記録を1行追加するだけで運用が完了します。
- **特記事項のみ記述:** 課題提出や勤怠に問題がなく、特筆すべき行動もなければ、「特記事項」の欄は「特になし」で構いません。負担をかけずに継続することを最優先してください。
- **客観的事実を記録:** 記録する内容は、引き続き「具体的な行動」や「数字」を意識してください。

### このフォーマットの運用方法

1.  **初期設定:**
    *   上記のテンプレートをコピーして、受講生ごとにmdファイルを作成します。
    *   テーブルの最終行にある `202X-XX-XX` の日付を、記録を開始する日付に書き換えてください。

2.  **日々の記録（毎日の作業）:**
    *   **日付、勤怠、課題提出、完成度/品質**の4つのセルをまず埋めます。
    *   何か特筆すべき出来事があった場合のみ、**「特記事項」**のセルを埋めます。
        *   まず**【カテゴリ】**を上記のカテゴリ一覧から選びます。
        *   次に、観察された**行動**、それに対するあなたの**指導**、本人の**反応**を `→` や記号でつなげて簡潔に記述します。
    *   一日の記録が終わったら、テーブルの最終行に **新しい空の行 `| | | | |` を追加**しておくと、翌日の追記がスムーズになります。

### 「特記事項」欄の記述例

-   **ネガティブな例:**
    `【質問力】講義で説明済みの内容を質問 → (指導) まず資料の該当箇所を読むよう指示 → (反応) 無言で資料を開くも、不満げな様子。`

-   **ポジティブな例:**
    `【主体性】初めて自分から他の受講生に不明点を質問し、解決していた。`

-   **問題が複数あった日の例:**
    `①【勤怠】午前中の休憩から5分遅れて戻る。②【注意力】課題の要件定義を読み飛ばし、指摘するまで気づかず。`
